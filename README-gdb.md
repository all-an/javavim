1 run cpp tests:

```
bash test_cpp.sh
```

2 run gdb for a specified file

```
gdb build/JavavimCppTest

```

Basic GDB Commands:

    Run the program: run
    Set a breakpoint: break <file>:<line_number> (e.g., break JavavimTest.cpp:10)
    Step through the code: step (steps into functions) or next (steps over functions)
    Print the value of a variable: print <variable_name> (e.g., print myVariable)
    Continue execution: continue
    Quit GDB: quit
